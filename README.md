# Jquery Snippets for [Atom](http://atom.io)

## Install

Go to `Atom > Preferences...` then search for **Jquery Snippets** in Packages tab.

## Development

```sh
$ cd ~/.atom/packages
$ git clone https://github.com/mojouy/atom-jquery.git
$ cd atom-jquery-snippets
$ apm install
$ apm link
```

## Features

### $

```javascript
$('$1')$0
```
